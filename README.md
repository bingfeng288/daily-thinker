# Daily Thinker - 每日想你小程序

一个简单的纯前端网页应用，记录每天想念次数的小程序。

## 功能特性
- **三个计数器**: 亲亲、抱抱、睡觉
- **本地存储**: 使用浏览器 localStorage 保存数据
- **日历展示**: 显示每日统计数据
- **无需服务器**: 可以直接部署到 GitHub Pages

## 使用方法
1. 打开 `index.html` 文件
2. 点击三个按钮增加想念次数
3. 数据会自动保存在浏览器本地
4. 查看日历统计

## 部署到 GitHub Pages
1. 在 GitHub 仓库中选择 Settings
2. 找到 Pages 选项
3. 选择 Source 为 `main` 分支
4. 等待几分钟，应用就会上线

## 项目文件
- `index.html` - 主页面文件
- `README.md` - 说明文档

## 技术特点
- 纯 HTML/CSS/JavaScript
- 无需后端服务器
- 数据存储在浏览器本地
- 响应式设计（适配手机和电脑）
